# Tree App

### Overview:
Tree App is a real time application, making posible that all user see any changes made immediately accross browser without refreshing or polling. This app is part of the Passport Inc's challenge.

### Technologies:

  ##### Front-End
  - Angular 6 
  - Angular Material
  - CSS
  - sockect.io-client
  
  ##### Back-End
  - Node.js
  - Mongoose
  - sockect.io-server
 
  ##### Database
  - MongoDB
